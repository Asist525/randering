🤖
Gemini AI · LangGraph 파이프라인
Google Gemini Flash를 LangGraph 상태 머신으로 오케스트레이션합니다. PM → SA → Agile로 이어지는 파이프라인이 일관성 있는 산출물을 보장하며, 각 단계가 이전 결과물을 참조해 설계의 무결성을 유지합니다.
👥
팀 협업 · 스냅샷
PM · 엔지니어 · 뷰어 역할로 접근 권한을 분리합니다. 분석 결과를 팀 스냅샷으로 발행하면 모든 팀원이 동일한 설계 기반을 공유합니다.
🔗
GitHub 완전 연동
OAuth 로그인, PR 웹훅 자동 수신, Issue 자동 발행까지 GitHub 워크플로우에 자연스럽게 통합됩니다.
💬
아이디어 채팅
AI와 대화하며 요구사항을 구체화합니다. 번복된 결정은 자동으로 필터링되어 최종 확정 사항만 파이프라인에 전달됩니다.
🔒
로컬 실행 · 보안
Python 런타임이 앱에 내장되어 별도 설치가 없습니다. 분석 데이터는 로컬 SQLite에 저장되며 외부로 전송되지 않습니다.
🔄
자동 업데이트
새 버전이 출시되면 백그라운드에서 자동 다운로드 후 앱 재시작 시 즉시 적용됩니다. 항상 최신 버전을 유지합니다.
